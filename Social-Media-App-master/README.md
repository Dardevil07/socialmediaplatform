# Social-Media-App
Full stack, fully-featured social media application using React, Firebase, Redux, Express, and Material-UI.  
-------------------------------------------------
# Running Frontend side locally:

To run this app:
## 1: Install packages

run `npm install`

## 2: Run project

run `npm start`

## 3: Open it

go to [http://localhost:3000](http://localhost:3000)
